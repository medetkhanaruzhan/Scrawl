#Scentra 

Perfume Recommendation Platform

Overview

Scentra is a web application designed to help users find perfumes that better match their personal preferences.

Instead of browsing large catalogs, users provide a small amount of information about their tastes and lifestyle, and the system suggests suitable fragrances.

The goal of the project is to make perfume selection more personalized and intuitive.

⸻

Concept

The application is based on the idea that fragrance choice depends not only on smell preferences, but also on a person’s mood, habits, and environment.

Scentra takes into account:
	•	preferred scent notes (e.g. vanilla, citrus, woody, floral)
	•	music taste and general mood
	•	favorite movies or aesthetic preferences
	•	climate and season
	•	budget and possible sensitivities

These factors are combined to generate recommendations that feel more relevant to the user.

⸻

Features
	•	Personalized perfume recommendations
	•	Simple preference form (quiz)
	•	Filtering by scent notes and budget
	•	Climate-based suggestions
	•	Basic authentication (login system)
	•	Viewing perfume information

⸻

Technologies Used
	•	Frontend: Angular
	•	Backend: Django REST Framework
	•	Authentication: JWT
	•	Database: SQLite (development)
